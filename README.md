# Task Manager App

A simple full-stack task management application built with React and Node.js.

## Quick Start

### Prerequisites
- Node.js installed
- MongoDB installed and running

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/usman-dev56/Task-Manager-App.git
   cd task-manager-app

## Project Structure
   task-manager-app/
├── frontend/     # React app
├── backend/      # Node.js API
└── README.md

---------------------------------------------------------------------------------------------------------------
**Features**
   - User registration and login

  - Create, edit, delete tasks

  - Mark tasks as complete/incomplete

  - Set task priorities (High, Medium, Low)

  - Filter tasks (All, Active, Completed)

  - Responsive design
---------------------------------------------------------------------------------------------------------------

** Tech Stack **

Frontend:
  React + Vite
  Bootstrap

Backend:
  Node.js
  Express
  MongoDB
  JWT Authentication

**  How to Use **

  Register a new account
  Login with your credentials
  Add tasks using the form
  Edit, complete, or delete tasks as needed
  Use tabs to filter tasks


  **API Endpoints**
 - POST /api/auth/register - Register user
 - POST /api/auth/login - Login user
 -  GET /api/tasks - Get tasks
 - POST /api/tasks - Create task
 - PUT /api/tasks/:id - Update task
 - DELETE /api/tasks/:id - Delete task
